服务器的注意事项：

  1. 远程服务器不允许关机，只能重启
  2. 重启时应关闭所有服务
  3. 不要在服务器访问的高峰运行高负载命令(杀毒，查找，系统备份等)
  4. 远程配置防火墙（过滤器，过滤掉ip，端口等）的时候不要把自己踢出服务器
  5. 合理的密码规范并且定期更新
  6. 合理分配权限，越小越好，系统启动的服务越少越好
  7. 定期备份重要数据和日志，不要把鸡蛋放在同一个篮子里

权限管理命令

only root and owner can modify the file 权限
chmod [{ugoa} {+-=}{rwx}][file or directory]] 『可以同时赋予多个权限，之需要使用“，”分割就可以了』
chmod [mode = 421] 『r=4,w=2,x=1』 [filename or directory]
when modify 上层目录权限，不会改变下层目录权限，如果需要全部改变子目录下所有文件的权限，可加上选项 -R

针对文件，针对目录

  1. r 读取权限 可以查看文件内容 cat/more/less/tac/head/tail 可以列出目录中的内容
  2. w 写入权限 可以修改文件内容 可以在目录中创建删除权限
  3. x 执行权限 可以执行文件 可以进入目录

chown [username] [filename or directory] 只有root才能更该文件所有者

umask [-S] {文件的默认缺省权限,-S以rwx显示} 【缺省创建文件的权限默认没有x权限】
